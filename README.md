🌸 PlantBot — Карта красивых цветов города

Telegram-бот для совместного сбора и поиска красивых цветочных мест в городе! 🌺

Пользователи отмечают места с цветами (📍 + 📸 + описание), другие находят ближайшие точки в радиусе 1 км.

✨ Возможности

<p><strong>➕ Добавить цветы</strong>	Локация → Фото → Описание → Сохранение в Supabase</p>
<p><strong>🔍 Найти рядом	</strong>    Ближайшие цветы в 1 км с картами и фото</p>
<p><strong>📊 Статистика	</strong>    Количество мест и пользователей</p>
<p><strong>🗺️ Карты	 </strong>       Google Maps + OpenStreetMap ссылки</p>
<p><strong>📱 Главное меню	 </strong>   Удобные кнопки вместо команд</p>
